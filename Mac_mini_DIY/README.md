# Mac mini DIY

> Mac mini DIY 主要指的是利用**Mac mini**的机壳DIY组装成迷你主机。

因为**Mac mini**的空间有限，通常都需要搭配超薄主板+配套的定制散热；

考虑到不同的 **主板 + 定制散热** 组合搭配会形成各种各样的装机方案；

这里就整理分享各种适用于Mac mini装机的主板、CPU、散热等硬件，和各种成品方案供大家参考。
